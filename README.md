# Game_Caro
Game caro môn lập trình Java
# Các chức năng
- chơi với máy (ở đây máy sẻ được sử dụng thuật toán min max cắt tỉa anpla beta)
- chơi với người (chơi cùng máy)
- chơi với người (chơi khác máy chung mạng LAN)
# Kiến thức
- Sử dụng giao thức socket.
- Lập trình đa luống.
- Client server.
